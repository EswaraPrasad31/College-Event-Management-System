# 🎓 CVR College Event Management System

A responsive **College Event Management System** developed for **CVR College of Engineering** using **HTML, CSS, JavaScript, and Bootstrap 5**. The portal enables students to explore upcoming events, view schedules, browse event galleries, register for events, and contact event coordinators through an intuitive user interface.

---

## ✨ Features

### 🏠 Home Page (`index.html`)

* Attractive hero section with college branding
* Featured events carousel
* Event statistics and highlights
* Quick access to registration

### 🎉 Events Page (`events.html`)

* Displays upcoming college events using Bootstrap cards
* Event details including date and venue
* Registration modal for event enrollment

### 📅 Event Schedule (`schedule.html`)

* Responsive event timetable
* Built using Bootstrap tables
* Supports striped and hover effects

### 🖼️ Gallery (`gallery.html`)

* Event photo showcase
* Responsive image grid layout
* Interactive card-based gallery design

### 📝 Registration (`registration.html`)

* Student event registration form
* Form validation using Bootstrap components
* Success message displayed after submission

### ❓ FAQ (`faq.html`)

* Accordion-based Frequently Asked Questions
* Easy-to-use expandable sections

### 📞 Contact (`contact.html`)

* Contact form for inquiries
* Event coordinator information
* Social media links and college contact details

## 🚀 How to Run

### Method 1: Open Directly

1. Download or clone the project.
2. Ensure all files and folders remain in their original structure.
3. Open `index.html` in any modern web browser.

### Method 2: Use a Local Server (Recommended)

For the best experience, run the project using:

* VS Code Live Server Extension
* XAMPP
* Any static HTTP server

This helps avoid asset-loading and path-related issues.
